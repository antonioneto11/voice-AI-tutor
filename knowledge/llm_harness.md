# LLM Harnesses

## What an LLM harness is
An LLM harness is the layer you use to run, test, compare, and monitor language model behavior in a repeatable way. It wraps prompts, tools, inputs, outputs, metrics, and experiment settings so teams can evaluate changes without guessing.

## Why teams use a harness
A harness helps teams avoid one-off prompt testing. It gives structure for regression tests, side-by-side model comparisons, prompt versioning, latency tracking, and failure review.

## Example
In a fintech support product, a harness might run 200 customer questions against two prompt versions, score both with a rubric, compare latency and refusal behavior, and flag which version is safer for launch.
